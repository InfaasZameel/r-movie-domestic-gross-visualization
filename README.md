# 🎬 Movie Domestic Gross % by Genre — R Visualization

A data visualization project built in R, recreating a bubble chart that analyzes the **domestic gross percentage** of movies across different genres and studios.

## 📊 Chart Description

The visualization is a **bubble chart** that shows:
- **X-axis**: Movie Genre (Action, Adventure, Animation, Comedy, Drama)
- **Y-axis**: Gross % in the US market
- **Bubble Size**: Movie Budget (in $M)
- **Color**: Studio (Buena Vista, Fox, Paramount, Sony, Universal, WB)

## 🛠️ Tools & Libraries Used

- **Language**: R
- **Libraries**: `ggplot2`

## 📁 Files

| File | Description |
|------|-------------|
| `movie_gross.R` | Main R script to generate the chart |
| `Movie-Ratings.csv` | Dataset used for the visualization |
| `chart_output.png` | Final chart output |

## 🚀 How to Run

1. Clone the repository:
   ```bash
   https://github.com/InfaasZameel/r-movie-domestic-gross-visualization.git
   ```
2. Open `movie_gross.R` in RStudio
3. Install dependencies if needed:
   ```r
   install.packages("ggplot2")
   ```
4. Run the script — the chart will be generated.
